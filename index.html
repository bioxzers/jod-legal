โดยย่อ: Jod เก็บข้อมูลการเงินไว้ในอุปกรณ์ของคุณเท่านั้น ไม่มีบัญชีผู้ใช้ ไม่ขายข้อมูล ไม่ส่งข้อมูลการเงินไปยังบุคคลที่สาม

1. ข้อมูลที่เราจัดเก็บ

จัดเก็บในอุปกรณ์ของคุณ (ไม่ส่งออก)

ข้อมูลต่อไปนี้ถูกจัดเก็บในอุปกรณ์และไม่ถูกส่งไปยัง Jod หรือบุคคลที่สาม:

  • รายการธุรกรรม — รายรับ รายจ่าย ยอดเงิน หมวดหมู่ วันที่ หมายเหตุ แท็ก
  • ข้อมูลบัญชี — ชื่อ ประเภท ยอดคงเหลือ วันครบกำหนดชำระบัตรเครดิต วงเงิน (ไม่มีเลขบัตร)
  • ชื่อที่แสดง — ชื่อที่คุณตั้งใน Settings (ไม่บังคับ ไม่ใช่ตัวตนจริง)
  • ชื่อสถานที่ — ชื่อสถานที่แนบกับธุรกรรม และปั๊มน้ำมันในฟีเจอร์บันทึกน้ำมัน
  • ข้อมูลยานพาหนะ — ชื่อรถ ประเภท ทะเบียน เลขไมล์ (สำหรับฟีเจอร์ Fuel Log)
  • บันทึกการเติมน้ำมัน — ประเภทน้ำมัน ราคา ลิตร เลขไมล์ ชื่อปั๊ม
  • การตั้งค่า — ธีม ภาษา สกุลเงินหลัก การแจ้งเตือน

ซิงค์ผ่าน iCloud (หากเปิดใช้งาน)

หากคุณเปิดใช้งาน iCloud ข้อมูลข้างต้นจะซิงค์ผ่าน Apple CloudKit ซึ่งอยู่ภายใต้ นโยบายความเป็นส่วนตัวของ Apple คุณสามารถลบข้อมูล iCloud ได้จาก Settings ในแอพ

ส่งไปยัง Firebase

เพื่อปรับปรุงแอพ Jod ส่งข้อมูลต่อไปนี้ไปยัง Google Firebase:

  • Crash Reports (Crashlytics) — ข้อมูล crash แบบไม่ระบุตัวตนเพื่อช่วยแก้บัก
  • Analytics Events — สถิติการใช้ฟีเจอร์แบบรวม ไม่มีข้อมูลส่วนตัว
  • Remote Config — ดึง URL แหล่งข้อมูลภายนอก (อัตราแลกเปลี่ยน ราคาน้ำมัน หวย) เพื่อให้อัพเดทได้โดยไม่ต้องออกเวอร์ชันใหม่ ไม่มีข้อมูลส่วนตัวเกี่ยวข้อง

ข้อมูลการเงินของคุณไม่ถูกส่งไปยัง Firebase เลย

ข้อมูลภายนอกที่ดึงสำหรับ Gimmick Tools

ฟีเจอร์บางอย่างดึงข้อมูลสาธารณะจากแหล่งภายนอก ข้อมูลเหล่านี้ไม่ใช่ข้อมูลส่วนตัวของคุณและไม่เกี่ยวข้องกับบัญชีหรือธุรกรรมของคุณ:

  • ราคาน้ำมัน (Bangchak) — ดึงข้อมูลราคาน้ำมันสาธารณะผ่าน Cloudflare Workers (อัพเดทอัตโนมัติทุก 10 นาที) เพื่อแสดงราคาวันนี้และพรุ่งนี้
  • อัตราแลกเปลี่ยน — ดึงจาก API ของธนาคารแห่งประเทศไทย (BOT) วันละครั้ง เพื่อแปลงสกุลเงินในธุรกรรม
  • ผลสลากกินแบ่ง — ดึงข้อมูลรางวัลสาธารณะสำหรับฟีเจอร์ตรวจหวย

ไม่มีข้อมูลส่วนตัวถูกส่งระหว่างการดึงข้อมูลภายนอกเหล่านี้

Privacy Policy

Effective: April 5, 2026 · Version 2.0

TL;DR: Jod stores your financial data locally on your device only. No user accounts, no data selling, no financial data sent to third parties.

1. Data We Collect

Stored Locally on Your Device

The following data is stored on your device and not transmitted to Jod or third parties:

  • Financial transactions — income, expenses, amounts, categories, dates, notes, tags
  • Account information — name, type, balance, credit card due dates, credit limits (no card numbers)
  • Display name — name you set in Settings (optional, not your real identity)
  • Location names — place names attached to transactions and fuel stations (requires permission)
  • Vehicle information — vehicle name, type, license plate, odometer readings (for Fuel Log feature)
  • Fuel log entries — fuel type, price, liters, odometer, station name
  • Preferences — theme, language, base currency, notification settings

Synced via iCloud (if enabled)

If you enable iCloud, the above data syncs through Apple CloudKit, governed by Apple's Privacy Policy. You can delete your iCloud data from within the app's Settings.

Sent to Firebase

To improve the app, Jod sends the following to Google Firebase:

  • Crash Reports (Crashlytics) — anonymous crash data to help fix bugs
  • Analytics Events — aggregate feature usage statistics, no personal data
  • Remote Config — fetches external data source URLs (exchange rates, oil prices, lottery) to allow updates without requiring a new app release. No personal data is involved.

No financial data is ever sent to Firebase.

External Data Fetched for Gimmick Tools

Some features retrieve publicly available data from external sources. This data is not your personal data and is unrelated to your accounts or transactions:

  • Oil Prices (Bangchak) — public oil price data fetched via Cloudflare Workers (updated automatically every 10 minutes). Used only to display current and tomorrow's oil prices.
  • Exchange Rates — fetched from the Bank of Thailand (BOT) API once daily. Used to convert currencies in transactions.
  • Lottery Results — public prize data fetched for the Lottery Checker feature.

No personal data is transmitted during any of these external data fetches.
